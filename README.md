# YongTauFoo

Using datasets from [Kaggle](https://www.kaggle.com/), this project analyses and visualises gaming industry trends to propose attributes that a game should have to break into the popular market.

### Datasets

[Video Games Rating By 'ESRB'](https://www.kaggle.com/datasets/imohtn/video-games-rating-by-esrb)
* [Training dataset](datasets/esrb_rating_training.csv)
* [Test dataset](datasets/esrb_rating_test.csv)

[Video game sales](https://www.kaggle.com/datasets/gregorut/videogamesales)
* [Dataset](datasets/video_game_sales.csv)
